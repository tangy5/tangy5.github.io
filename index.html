<!-- Main container of card -->
<div class="c-card__item">
<!-- Need to set card ratio - you may edit it in css, but do not delete -->
	<div class="c-card__sizer"></div>
	<div class="c-card__inner">
	<!-- Icon stays the same in modal -->
		<div class="c-card__icon">
			<i class="icofont icofont-earth"></i>
		</div>
		<!-- Header stays the same in modal -->
		<h4 class="c-card__header">Ecology Problems</h4>
		<!-- Content will be replased in modal ... -->
		<div class="c-card__content js-card-content">
			<p class="c-card__text">Lorem ipsum dolor sit amet.</p>
		</div>
		<!-- ...replaced by this modal content -->
		<div class="c-card__modal-content js-card-modal-content">
			<p class="c-card__text">Lorem ipsum dolor sit amet, consectetur adipisicing elit mollit anim id est laborum. .</p>
			<p class="c-card__link-caption">Check my projects:</p>
			<a class="c-card__link" href="#">Investigation of rabbit’s neuronal reaction</a>
			<a class="c-card__link" href="#">What is the main factor of peoples agression?</a>
		</div>
	</div>
	<!-- This button toggles modal content and closes it -->
	<div class="c-card__button js-toggle-modal">
		<button class="c-card__button-field" type="button"></button>
	</div>
</div>

Editing source code

First of all you need to prepare your environment.
  1. Download and install latest stable version of node.js and npm: https://nodejs.org
  2. Install gulp: http://gulpjs.com/ 
    $ npm install --global gulp-cli
  3. Open the folder with project (the root folder, not ./src or ./dist) in terminal (or cmd) and run: 
    $ npm install
  4. Install bower dependencies: 
    $ bower install
  5. If everything is alright simply run: 
    $ gulp
Index page will automatically open in your browser. To navigate to another page, you may want to add page name after a slash. It may look like this:
http://localhost:3000/property-item.html
Gulp starts to watch your changes in source code and show them in your browser, you even do not need to reload page you are editing. To build production version, run following command in your terminal 
$ gulp build
If you need to edit source files, you will need to learn SASS - CSS preprocessor and Nunjucks - it is HTML template engine from Mozilla
